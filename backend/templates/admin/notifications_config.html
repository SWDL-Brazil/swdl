📧 Servidor de E-mail (SMTP)
⚠️ Atenção: E-mails educacionais (@portalsesisp.org.br) geralmente não aceitam mensagens de SMTPs externos (Gmail, Outlook.com, etc.). Você precisa usar o SMTP do próprio SESI ou do provedor educacional (Office 365 / Microsoft 365, se a escola usar).

Deve ser um e-mail do próprio domínio educacional para evitar bloqueio.
💬 WhatsApp

Configure a API do WhatsApp para enviar as credenciais por mensagem. Compatível com Evolution API, Twilio WhatsApp ou qualquer serviço que aceite POST JSON com campos number, message e api_key.

💡 Se não tiver uma API, o aluno ainda receberá as credenciais por e-mail (se configurado) e o admin pode ver a senha na tela de aprovação.
